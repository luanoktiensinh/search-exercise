# Guidelines

## Requirements

Clone all the functionalities of the input in [this link](https://just-an-input.vercel.app/?path=/story/example-input--primary)

### Props
- `placeholder`: Placeholder of the input
- `onSelectItem`: On click item handler

### Functions

- Handle 4 state
  - `initital`: show input with placeholder
  - `fetching`: fetching data, show input with a loader
  - `success`: feteched data successfully, show list item or text "No results" if there are no results
  - `error`: fetched data failed, show error message
- When click into an item, trigger the `onSelectItem` function
- Add a debounce 100ms when fetching data
- Handle race condition

## Rules
- Implement your functionalities inside the component `src/components/Input` between these pair of comments: 
```
// Your code start here
  ...
// Your code end here
```
- Do not modify any other existing components/files. Feel free to create new components if needed
- use `fetchData` and `debounce` functions from `src/utils`
- use `Loader` components from `src/components/Loader`. Feel free to create another loader if you like.
